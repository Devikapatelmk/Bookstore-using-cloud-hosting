# Bookstore-using-cloud-hosting
# Online Bookstore

An online bookstore web application built using **Flask**, **MongoDB Atlas**, and **HTML/CSS**. Users can browse books, register/login, and book their favorite books. Bookings are stored securely in a MongoDB database in the cloud.

---

## **Features**

- Browse available books with details (title, author, description, price).  
- User authentication: register, login, and logout.  
- Book a selected book (only available to logged-in users).  
- Booking history stored in **MongoDB Atlas**.  
- Responsive UI using HTML/CSS templates.  

---

## **Technologies Used**

- **Backend:** Python, Flask  
- **Database:** MongoDB Atlas (Cloud-hosted)  
- **Frontend:** HTML, CSS, Jinja2 templates  
- **Python Packages:**  
  - `flask` – Web framework  
  - `pymongo` – MongoDB client for Python  
  - `dnspython` – DNS support for MongoDB Atlas
